# Лінійна регресія через нормальні рівняння (NumPy)

Цей проект демонструє реалізацію алгоритму лінійної регресії "з нуля" за допомогою матричних операцій бібліотеки NumPy. 

Мета проекту — показати, як обчислюються коефіцієнти регресії за допомогою лінійної алгебри, та порівняти ефективність множинної лінійної регресії з простою моделлю на основі синтетичних даних.

## Опис моделей

У скрипті реалізовано та порівнюється дві моделі прогнозування підсумкової вартості (`total`):
1. **Множинна регресія:** `total ~ price + quantity + discount`
2. **Проста регресія:** `total ~ price`

## Математичне підґрунтя

Для знаходження оптимальних ваг (коефіцієнтів) моделі використовується **нормальне рівняння** (Normal Equation), яке дозволяє знайти аналітичне рішення без використання ітеративних методів (наприклад, градієнтного спуску):

$$\theta = (X^T X)^{-1} X^T y$$

де:
* $X$ — матриця ознак (з доданим фіктивним стовпцем одиниць для обчислення вільного члена $b_0$).
* $y$ — вектор значень цільової змінної.
* $\theta$ — шуканий вектор коефіцієнтів моделі.
* $X^T$ — транспонована матриця ознак.
* $(X^T X)^{-1}$ — обернена матриця.

## Вимоги (Dependencies)

Для запуску коду потрібен встановлений Python та бібліотека NumPy.
